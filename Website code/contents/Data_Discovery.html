Data Discovery


For our analysis we used "Instagram fake spammer genuine accounts" dataset. This dataset is available on Kaggle website.


There are two datasets i.e 'train.csv' which will be used to train the machine learning model over a subset of the original dataset and 'test.csv' to test the quality & evaluate whether it can generalize well to the new or unseen data of our model.

Importing Datasets

Train Dataset


It is highly responsible for the model's accuracy and prediction ability. It means that the better the quality of the training data, the better will be the performance of the model. The training data lets the ML algorithms learn how to make predictions for the given task. Training data is approximately more than or equal to 60% of the total data for an ML project.

Test Dataset


After training the dataset, it's time to test the model with test dataset. This dataset evaluates the performance of the model & ensures that it can generalize well with the unseen dataset. This is a well-organized dataset that contains data for each type of scenario model would be facing in the real world. Test dataset is approximately 20% of the total original data for an ML project.
